About Our Web Tool

The Web tool is divided into two main parts
1)Heart-> The FrontEnd 2)Soul-> The Backend
The FrontEnd --> The FrontEnd is made of HTML5, CSS, JS, Jquery and also uses Python to make RESTful calls to backend.
The Backend --> The Backend is made in Spring Boo using Java8 following all the proper design patterns and mave RESTful APIs for each scenario.
The Database --> The Tool also have a inMemory Database setUp to store keys and creds H2Database.
The Project is using AES Algorithm to encrypt files at the BE.

About The Algorithm

The AES algorithm (also known as the Rijndael algorithm) is a symmetrical block cipher algorithm that takes plain text in blocks of 128 bits and converts them to ciphertext using keys of 128, 192, and 256 bits. Since the AES algorithm is considered secure, it is in the worldwide standard.
